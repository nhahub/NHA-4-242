# 📌 Overview

Cardiovascular disease remains one of the leading causes of death worldwide. Early prediction can significantly improve diagnosis and treatment planning.

This project develops a complete Machine Learning workflow capable of predicting the presence of heart disease using patients' clinical information. The notebook demonstrates every stage of the data science lifecycle, from raw data exploration to building and evaluating predictive models.

---

# 🎯 Project Objectives

* Analyze the heart disease dataset.
* Discover meaningful insights through Exploratory Data Analysis.
* Clean and preprocess the data.
* Engineer useful features.
* Handle class imbalance using SMOTE.
* Train multiple classification models.
* Compare model performance.
* Select the best-performing model.
